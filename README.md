# 📖 Jom Belajar Tajwid

> A mobile learning application to help users learn and practice **Tajwid** (Quranic recitation rules) — built with Flutter for Android.

---

## 📱 About

**Jom Belajar Tajwid** is an Android mobile application designed to make learning Tajwid interactive, engaging, and accessible. The app combines structured learning modules with quiz-based practice to reinforce understanding of Quranic recitation rules.

---

## ✨ Features

| Feature | Description |
|--------|-------------|
| 📚 **Learning Modules** | Structured notes and lessons on Tajwid rules |
| 🎧 **Audio Pronunciation** | Listen to correct pronunciation of Arabic letters |
| ❓ **Quiz System** | Test your knowledge with Tajwid-based questions |
| ✅ **Answer Explanation** | Get detailed explanations for correct and incorrect answers |
| 🔖 **Bookmark** | Save important questions for future reference |
| 🚩 **Flag Toggle** | Mark questions for review |
| 📊 **Progress Tracking** | Monitor your learning progress over time |

## 🚀 Getting Started

### Prerequisites
- Flutter SDK installed
- Android Studio / VS Code
- Android Emulator or physical device

### Installation

```bash
# Clone the repository
git clone https://github.com/nrsuhailah/jom-belajar-tajwid.git

# Navigate to project directory
cd jom-belajar-tajwid

# Install dependencies
flutter pub get

# Run the app
flutter run
```
